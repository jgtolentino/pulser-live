- Docker & Docker Compose
- Anthropic API Key
- 16GB RAM minimum
- GPU recommended for visual analysis

#### Installation

```bash
# Clone repository
git clone https://github.com/your-org/jampacked-creative-intelligence.git
cd jampacked-creative-intelligence

# Set up environment
cp .env.example .env
# Edit .env with your API keys

# Start services

# Verify health

```
